# ALFA Bank 🚀

ALFA Bank is a modern, secure, and premium digital bank simulator. Built with a focus on high-fidelity UI/UX, it demonstrates a complete banking flow including PIX, payments, card management, and financial dashboards.

## ✨ Key Features

- **Dynamic Dashboard**: Real-time balance and transaction charts using Recharts.
- **Full PIX Flow**: Send, receive, and manage PIX keys.
- **Card Management**: Virtual card viewing and instant blocking/unblocking.
- **Simulated Backend**: 100% frontend architecture with LocalStorage persistence and simulated API latency.
- **Premium UI**: Built with Tailwind CSS and shadcn/ui for a professional look and feel.

## 🛠️ Technologies

- **Frontend**: React + Vite + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: React Context API
- **Charts**: Recharts
- **Validation**: Zod + React Hook Form

## 🚀 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Start development server**:
   ```bash
   npm run dev
   ```
3. **Open in browser**: `http://localhost:5173`
